## Notebook: `02_hartley_walsh_basico.ipynb`

Este notebook compara tres formas de reorganizar señales discretas:

- FFT
- Walsh–Hadamard
- Hartley

Su objetivo es mostrar que el análisis espectral no se agota en Fourier y que distintas bases ortogonales revelan distintas estructuras de una señal.

### Contenido
- implementación de Fast Walsh–Hadamard Transform;
- implementación de Discrete Hartley Transform a partir de la FFT;
- comparación sobre señales suaves y señales por bloques;
- reconstrucción truncada;
- discusión conceptual sobre familias ortogonales.Aquí vivirán los notebooks de exploración.
