# Monografía armónica

Proyecto para desarrollar una monografía sobre dinámica débilmente no lineal, Fourier, balance armónico, lifting y aplicaciones en sistemas eléctricos de potencia.

## Estructura

- `book/`: manuscrito LaTeX de la monografía
- `python-lab/`: laboratorio en Python para exploración numérica
- `examples/`: señales y ejemplos eléctricos
- `assistant/`: futuro asistente AI con tool calling
- `docs/`: documentación general del proyecto
- `scripts/`: scripts auxiliares

## Visión
Texto + laboratorio + asistente conversacional.
